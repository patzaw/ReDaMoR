# Check if two [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md) are identical

Check if two
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)
are identical

## Usage

``` r
identical_RelDataModel(x, y, ...)
```

## Arguments

- x:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- y:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- ...:

  additional parameters for
  [`identical_RelTableModel()`](https://patzaw.github.io/ReDaMoR/reference/identical_RelTableModel.md)

## Value

A logical: TRUE if the 2 models are identical

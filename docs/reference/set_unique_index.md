# Set table index uniqueness in a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

Set table index uniqueness in a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

## Usage

``` r
set_unique_index(x, tableName, fieldNames, unique)
```

## Arguments

- x:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- tableName:

  the name of the table to modify (a single character)

- fieldNames:

  the names of the fields composing the index

- unique:

  a logical value

## Value

A
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

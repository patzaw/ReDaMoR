# Remove an index from a table in a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

Remove an index from a table in a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

## Usage

``` r
remove_index(x, tableName, fieldNames)
```

## Arguments

- x:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- tableName:

  the name of the table to modify (a single character)

- fieldNames:

  the names of the fields composing the index

## Value

A
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

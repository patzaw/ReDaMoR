# Check if the object is a [RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md) matrix object

A matrix model is a special
[RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md)
object with 3 and only 3 fields: 2 of types 'row' and 'column' and the
3rd of your choice.

## Usage

``` r
is.MatrixModel(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is a
[RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md)
matrix object

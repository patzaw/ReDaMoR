# Update the display of a table of a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

Update the display of a table of a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

## Usage

``` r
update_table_display(
  x,
  tableName,
  px = NULL,
  py = NULL,
  color = NULL,
  comment = NULL
)
```

## Arguments

- x:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- tableName:

  the name of the table to modify (a single character)

- px:

  the position of the table: x value

- py:

  the position of the table: y value

- color:

  the color of the table

- comment:

  a table description/comment

## Value

A
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

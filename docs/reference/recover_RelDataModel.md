# Recover an autosaved [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

Recover an autosaved
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

## Usage

``` r
recover_RelDataModel(name = NA)
```

## Arguments

- name:

  The name of the autosaved
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)
  to bring back. Available autosaved
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)
  can be listed using the
  [`list_autosaved_RelDataModel()`](https://patzaw.github.io/ReDaMoR/reference/list_autosaved_RelDataModel.md).
  If NA (default) the latest model is returned.

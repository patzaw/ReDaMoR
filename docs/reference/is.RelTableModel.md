# Check if the object is a [RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md) object

Check if the object is a
[RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md)
object

## Usage

``` r
is.RelTableModel(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is a
[RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md)
object

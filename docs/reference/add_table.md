# Add a table to a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

Add a table to a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

## Usage

``` r
add_table(x, newTable)
```

## Arguments

- x:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- newTable:

  the name of the new table or a
  [RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md)

## Value

A
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

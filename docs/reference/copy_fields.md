# Copy fields from one table to another in a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

Copy fields from one table to another in a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

## Usage

``` r
copy_fields(x, from, to, fields)
```

## Arguments

- x:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- from:

  the name of the table from which the fields are taken

- to:

  the name of the table to which the fields are copied

- fields:

  the names of the fields to copy

## Value

A
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

# Check if the object is a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md) object

Check if the object is a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)
object

## Usage

``` r
is.RelDataModel(x)
```

## Arguments

- x:

  any object

## Value

A single logical: TRUE if x is a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)
object

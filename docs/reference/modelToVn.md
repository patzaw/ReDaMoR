# VisNetwork representation of a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md) object

VisNetwork representation of a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)
object

## Usage

``` r
modelToVn(
  model,
  color = "lightgrey",
  border = "black",
  highlightBorder = "orange"
)
```

## Arguments

- model:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- color:

  default table background color

- border:

  border color (single character)

- highlightBorder:

  color of highlighted borders

  Internal function

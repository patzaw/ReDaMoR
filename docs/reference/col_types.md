# Get the types of the columns of a [RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md) object

Get the types of the columns of a
[RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md)
object

## Usage

``` r
col_types(x)
```

## Arguments

- x:

  a
  [RelTableModel](https://patzaw.github.io/ReDaMoR/reference/RelTableModel.md)
  object

## Value

A col_spec object with the type of each column

# Normalize type names

Normalize type names

## Usage

``` r
norm_type_ref(x, typeRef, ignore.case = TRUE)
```

## Arguments

- x:

  a character vector to normalize

- typeRef:

  a character vector of length one: the type reference
  ([list_type_ref](https://patzaw.github.io/ReDaMoR/reference/list_type_ref.md))

- ignore.case:

  should case be ignored (default: TRUE)

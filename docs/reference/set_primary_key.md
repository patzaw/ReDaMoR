# Set the primary key a table in a [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

Set the primary key a table in a
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

## Usage

``` r
set_primary_key(x, tableName, fieldNames)
```

## Arguments

- x:

  a
  [RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)

- tableName:

  the name of the table to modify (a single character)

- fieldNames:

  the names of the fields to include in the primary key

## Value

A
[RelDataModel](https://patzaw.github.io/ReDaMoR/reference/RelDataModel.md)
